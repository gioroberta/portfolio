Sobre mim

Sou desenvolvedora de software em formação, cursando Análise e Desenvolvimento de Sistemas, com foco em back-end e no desenvolvimento de APIs REST. Possuo experiência prática com JavaScript e TypeScript, utilizando NestJS, além de integração com bancos de dados relacionais (SQL).

Atuo com lógica de programação, orientação a objetos, modelagem de dados, versionamento com Git e organização de código seguindo boas práticas. Tenho vivência na estruturação de aplicações back-end, tratamento de erros, validação de dados e controle de fluxo de requisições.

Também possuo experiência com testes (principalmente testes unitários), noções de segurança de aplicações como autenticação e autorização com JWT, além de práticas básicas de deploy e publicação de aplicações, compreendendo o ciclo de desenvolvimento do código até a entrega em produção.
